## How to Use

1. **Duplicate this template** for each client
2. **Fill in all bracketed fields**
3. **Save as PDF** (Invoice_EVOL-2026-XXX_ClientName.pdf)
4. **Send via WhatsApp/Email**
5. **Follow up** if payment not received in 3 days
6. **Store copy** in client folder

## Invoice Numbering Format

`EVOL-INV-2026-[Sequential Number]`

Examples:
- EVOL-INV-2026-001 (First client)
- EVOL-INV-2026-002 (Second client)
- EVOL-INV-2026-003 (Third client)

Maintain sequential order for accounting.
