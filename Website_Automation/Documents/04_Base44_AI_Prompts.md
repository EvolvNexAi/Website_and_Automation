**Use:** Copy-paste these prompts into Base44 Builder Chat to generate client websites

## How to Use These Prompts

1. **Fill in brackets** `[ ]` with client info from onboarding form
2. **Paste into Base44** Builder Chat
3. **Review** generated site
4. **Tweak** via Discussion Mode if needed
5. **Publish** to client's subdomain

## Prompt Template 1: Clinic/Doctor Website

```
Build a medical clinic website for [Clinic Name].

INFO:
- Clinic: [Name] | Doctor: [Name, Qualifications]
- Speciality: [e.g., Dental, Dermatology] | Experience: [X] years
- Location: [City, Area] | Phone: [Number] | WhatsApp: [Number]
- Hours: [Mon-Sat 10am-7pm] | Email: [Email]

PAGES:
1. HOME - Hero: "[Speciality] Care in [City]", CTAs (Book/Call), Services preview (3-4), Why Choose Us, Testimonials, Map preview
2. ABOUT - Doctor photo, Bio, Qualifications, Memberships, Care approach
3. SERVICES - 6-10 services as cards (Name + 2-line desc, Book button)
4. CONTACT - Address + Maps, Phone, WhatsApp (floating), Email, Hours, Contact form
5. GALLERY - Clinic/equipment photos (optional)

DESIGN:
- Colors: Blue/White/Gray (medical, trustworthy)
- Fonts: Clean sans-serif (Poppins/Inter)
- Style: Modern, minimal, mobile-first
- Trust signals: Certifications, council registration

FEATURES:
- Floating WhatsApp button (bottom-right)
- Click-to-call, mobile-responsive, fast loading
- SEO: "[speciality] doctor in [city]"
- Google Maps embed, contact form, social links

Generate full site with all pages and features.
```

## Base44 Workflow Tips

### After Generating:
1. **Review in Mobile View** - Test on mobile preview
2. **Check All Links** - Ensure CTAs work
3. **Test Forms** - Submit test inquiry
4. **WhatsApp Button** - Verify number is clickable
5. **SEO Check** - Meta title/description set
6. **Favicon** - Add client logo as favicon
7. **Connect Domain** - Use Base44 custom domain feature

### Discussion Mode Commands:
```
"Change the hero section color to lighter blue"
"Make the WhatsApp button larger and more prominent"
"Add a testimonials section to the home page"
"Reduce spacing between service cards"
"Use Poppins font throughout"
```

### Export/Handoff:
- Export code if client wants (Base44 Builder plan+)
- Keep hosting on Base44 for maintenance revenue
- Set up client training session
